源代码：编程软件种打开的代码(需编译后才可运行)
编译：把代码转换为2进制文件
集成开发环境（IDE）：使用的编程软件
符号常量:在源代码中看到的量(如12312)

compile 		        当前文件编译
build (or make)         编译所有
link 		            把源代码和库连接起来
run (excute) 	        运行
STL                     标准模板库
MI                      多重继承
ABC                     抽象基类
接口                     函数
方法                     函数
函数/类 签名             返回类型 函数名(参数类型列表) 限定符
迭代器                  广义指针
子序列                  字符串 S 的 子序列 是从 S 中将若干元素提取出来并不改变相对位置形成的序列
